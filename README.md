# DIP — Buy the Dip. Literally. 🟢

An instant-play browser game for the Orynth Game Cup. Dive into the red dump waves, thread the green dip, slingshot to the moon. Ticker: **$DIP**.

Single self-contained HTML file. Canvas 2D + procedural WebAudio. No build, no assets, no dependencies.

**Play:** open `index.html` or the GitHub Pages URL.
